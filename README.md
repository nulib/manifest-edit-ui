# manifest-edit-ui
Basic UI for IIIF Manifest editor intended for the Maktaba project.

## Usage

**Production Build**
```
npm run build
```
- outputs to `dist/`

**Local Devlopment**
```
npm run dev
```
- Local:  http://localhost:5173/

**Configuration**

- rename the `.env.example` file to `.env` and replace with your own values.
